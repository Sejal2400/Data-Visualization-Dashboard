-----Introduction------

----------Data Visualization Dashboard -----------

Data visualization is a way of presenting data in a visual form to make it easier to understand and analyze.
 Data dashboards are a summary of different, but related data sets, presented in a way that makes the related information easier to understand.

 Skills used:
 Python ,Django Framework,Chartjs,datatable,Html,Bootstrap


 ---To start this project----
 If your project is already in an existing python3 virtualenv first install django by running

         $ pip install django

Activate the virtualenv for your project.

    $ ven_name/Scripts/Activate
    

        cd projectname
Then simply apply the migrations:

        $ python manage.py migrate
You can now run the development server:

        $ python manage.py runserver
